# WERRA
Webpack+ECMAScript6+React+Redux+Antd

### 简介
使用react开发了一段时间了，但都是开发业务，不太了解配置和依赖，自己在业务时间折腾一下，会有所收获。

### 安装启动
+ git clone https://github.com/viviarui/WERRA.git
+ cd WERRA
+ npm install
+ npm run start
